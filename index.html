Enquête Métier

Pouvez-vous me décrire votre métier et me préciser avec quels outils vous travaillez ?

Je suis développeur web / développeur back-end.
Je travaille principalement sur la partie PHP d’application web ou de site internet.
Il m’arrive aussi de toucher au front-end avec du Javascript/JQuery ainsi que du HTML/CSS.
Je travaille avec les principaux framework PHP que sont Symfony et Laravel.
L'objectif principal de mon métier est de faire le lien entre les données stockées en base de données (principalement MySql et PostGreSql) et l’interface graphique utilisée par les utilisateurs.

Comment se déroule une journée type ?

On commence généralement la journée avec une réunion de 15 minutes pour permettre le suivi de l'avancée de l'équipe.
On a ce ce qu’on appelle un "backlog" avec une liste de tâche à effectuer (amélioration, création de nouvelles fonctionnalités, correction de bug).
Je traite les tâches une par une en fonction de la priorité définie par le chef de projet.
Une fois une tâche finie je la mets en attente de relecture par d’autres développeurs de l’équipe.
Quand mon code est validé, il est ajouté à l’application sur laquelle je travaille.
Bien entendu je dois aussi relire le code des autres développeurs.

Pourquoi avoir choisi ce métier ?

J’ai choisi ce métier car j’ai commencé mes études dans le graphisme et j’étais frustré de dessiner des sites internet sans pouvoir les faire fonctionner.
J’ai donc appris à coder et je suis resté dans ce milieu qui me convenait bien.
